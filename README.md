# MMORPG 分鑽系統

支援多款遊戲的分鑽系統，提供開單分鑽、銀行系統、拍賣系統、DKP系統等功能。

## 功能特色

- **團隊管理** - 創建管理團隊、成員管理、角色權限設定
- **BOSS 記錄** - 記錄首領擊殺、掉落物品管理
- **開單分鑽** - 自動建立分鑽單、成員分配、扣薪/不扣薪入帳
- **錢包系統** - 個人鑽石錢包、儲值/提領、交易明細
- **團隊公基金** - 團隊共同資金管理
- **拍賣系統** - 發起拍賣、競標出價、結標扣款
- **DKP 系統** - 貢獻點數管理、排行榜
- **會計報表** - 每月統計、成員貢獻排行

## 快速開始

```bash
# 安裝依賴
pip install -r requirements.txt

# 啟動服務
python app.py

# 打開瀏覽器
# http://127.0.0.1:5050
```

## 支援遊戲

天堂全系列 (M, 2M, W, 經典版)、HIT2、夜鴉 NIGHT CROWS、
ROM 王權之憶、波拉西亞戰記、權力之望、RAVEN2：渡鴉、洛汗2、血之繼承者

## 使用流程

1. 創建團隊 -> 設定遊戲與管理員
2. 新增成員 -> 加入血盟夥伴
3. 記錄BOSS擊殺 -> 自動建立分鑽單
4. 填寫掉落物品 -> 設定價值與分配方式
5. 分配鑽石 -> 選擇入帳方式完成分配
6. 管理拍賣 & DKP -> 分配稀有物品與計算貢獻
